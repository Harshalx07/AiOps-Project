🚀 AI-powered system for detecting anomalies in system logs using **Python and machine learning**.

---

## 📌 Overview

This project analyzes **system logs** using 🐍 Python and 🤖 machine learning techniques to detect anomalies such as:

🔐 Failed login attempts
🚫 Unauthorized access
⚠️ High CPU usage
💥 Service failures

The system processes log data, identifies abnormal patterns, and generates reports to help **DevOps teams monitor infrastructure health**.

---

## ⚡ Features

📄 Log parsing and analysis
🚨 Detection of suspicious system activity
❗ Identification of critical errors and warnings
🧠 Automatic anomaly detection using machine learning
📊 Generation of anomaly reports
📈 Visualization of log frequency patterns
